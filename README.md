# KakoiKids - Tienda de Ropa para Niños

Proyecto de sitio web e-commerce minimalista y funcional.

## Características

*   **Diseño Minimalista**: Paleta de colores suaves (Verde Salvia y Crema).
*   **Filtrado Funcional**: Filtra productos por Categoría (Bebé, Niño, Niña) y Talla.
*   **Carrito de Compras**:
    *   Agregar productos.
    *   Ver resumen en barra lateral.
    *   Cambiar cantidades.
    *   Eliminar items.
*   **Responsive**: Se adapta a diferentes tamaños de pantalla.

## Tecnologías

*   HTML5
*   CSS3 
*   JavaScript
